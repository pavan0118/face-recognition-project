# Face Recognition Project

This project performs real-time face recognition using Python, OpenCV, and the face_recognition library.

## Features
- Opens webcam
- Detects faces in real time
- Recognizes known face
- Displays the person's name on screen

## Technologies Used
- Python
- OpenCV
- face_recognition

## How to Run
1. Install required libraries:
   pip install opencv-python face-recognition

2. Place a known image inside:
   known_faces/pavan.jpg

3. Run the program:
   python main.py
